# versha password checker (c++)

---

# about

* simple password strength checker written in c++
* analyzes password security based on basic rules
* built as a beginner cybersecurity project

---

# features

* detects weak, medium, and strong passwords
* checks for:

  * uppercase and lowercase letters
  * numbers and special characters
  * minimum length
  * warns if password is common(from wordlist i got from @danielmiessler)

---

strength: strong :)
