# versha password checker (c++)

---

# about

* basic password security tool written in c++
* checks how strong a password is
* gives feedback and suggestions to improve it

---

# features

* detects weak, medium, and strong passwords
* shows detailed issues:

  * missing uppercase / lowercase
  * missing numbers or symbols
  * short length
* warns about common passwords (wordlist i got from @danielmiessler)
* suggests stronger passwords
* estimates crack time

---

# example

enter password: echo

password analysis:
strength: weak >:(

issues:

* too short
* missing uppercase letter
* missing number
* missing special character

suggested password:
Echo123!

estimated crack time: instantly (very weak)
